# awesome-python3-webapp

## python3 实验课

### 基础 `/lessons/basic/*.py`

1. python 文件以 `.py` 为后缀
2. python 两种运行方法
    - 交互模式 -> 命令行里敲 `$ python3` 一行一行地输入源代码，每输入一行就执行一行
    - 直接运行 -> 命令行里敲 `$ python3 {filename}` 一行一行地输入源代码，每输入一行就执行一行
3. 如何直接运行 python 文件
    - 文件第一行加 `#!/usr/bin/env python3`
    - 命令行运行 `$ chmod a+x {filename}` 修改执行权限
    - 命令行运行文件 `$ ./{filename}`
4. 输入输出
